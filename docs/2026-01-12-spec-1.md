## 需求分析

你希望在连接进入 Factory 时：
1. **自动启动** 项目的开发服务器
2. **输出当前项目的监控网页地址**

## 实现方案

### 使用 Factory Hooks 机制

在项目根目录创建 `.factory/hooks/on-session-start.md` 文件，配置会话启动时的自动行为：

```markdown
# 会话启动钩子

## 自动执行任务

1. 启动 Vite 开发服务器（后台运行）
2. 输出监控网页地址

## 执行命令

```bash
cd D:\Project\AIProject\AIView && npm run dev
```

## 监控网页地址

Vite 默认端口: **http://localhost:5173**
```

### 具体实现步骤

1. 创建 `.factory/hooks/` 目录
2. 创建 `on-session-start.md` 钩子文件，内容包含：
   - 后台启动 `npm run dev` 命令
   - 输出监控网页地址提示

### 预期效果

每次连接进入时，Factory 会：
1. 自动在后台启动 Vite 开发服务器
2. 显示提示信息：**监控网页地址: http://localhost:5173**
