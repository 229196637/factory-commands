## 技能提案：sce-api-reference

### 目的
当设计星火引擎底层API或需要查阅星火编辑器官方API文档时，提供系统化的查阅指南和快速参考。

### 位置
**Personal** (`C:\Users\unclefata\.factory\skills\sce-api-reference\`) - 因为这是跨项目通用的参考技能

### 技能目录结构
```
sce-api-reference/
├── SKILL.md              # 主技能文件：查阅流程和API概览
└── references.md         # 文档链接索引和API分类速查表
```

### SKILL.md 内容概要
1. **触发场景**：设计星火API、查阅官方文档、遇到 `base._xxx` 调用
2. **文档结构导航**：如何找到正确的API页面
3. **核心API速查**：
   - 服务端Lua API（单位、技能、Buff、事件等）
   - 客户端Lua API（UI、游戏、输入等）
   - Lua特殊限制和修改
4. **常用代码模式**：事件监听、单位属性操作等
5. **注意事项**：客户端/服务端差异、32/64位问题

### references.md 内容概要
- 官方文档入口链接
- API分类索引（按功能模块）
- 常见问题文档链接
- 开发者资源链接

### 不包含的内容
- 完整API文档复制（应直接查阅官方文档）
- 脚本自动化（文档是在线的，需要实时查阅）

### 为什么这样设计
1. **轻量级**：只提供导航和速查，不复制完整文档
2. **实用性**：聚焦于"如何快速找到需要的API"
3. **可维护**：文档链接变化时只需更新 references.md
